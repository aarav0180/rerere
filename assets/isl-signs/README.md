# ISL Alphabet Signs

This folder contains Indian Sign Language (ISL) alphabet hand sign images.

## Required Images

Place ISL hand sign images for each letter A-Z in this folder:
- A.png
- B.png
- C.png
- ... (through Z.png)

## Image Guidelines

- **Format**: PNG with transparent background
- **Size**: Recommended 200x200 pixels or larger
- **Content**: Clear hand gesture showing ISL sign for each letter
- **Background**: Transparent or white background
- **Quality**: High resolution for clear visibility

## Sources for ISL Signs

You can obtain ISL alphabet images from:
1. ISLRTC (Indian Sign Language Research and Training Centre)
2. Create custom images from ISL learning resources
3. Screenshot from ISL tutorial videos (with proper attribution)

## Fallback

If images are not available, the app will show placeholder images or use text labels.
